# Glossary

Every domain term used in the system design document must appear here.
Each term must be verified against industry standards (3+ sources).

## How to Use

When introducing a domain term in the document:
1. Check if it's already in this glossary
2. If not, search the web for the industry-standard definition
3. Find at least 3 corroborating sources
4. If fewer than 3 sources confirm the term, flag it for review
5. Add the term below with its definition and sources
